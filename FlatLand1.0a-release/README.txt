FlatLand requires Java 19: https://www.oracle.com/java/technologies/javase/jdk19-archive-downloads.html
You must then place the location of the java binary folder onto your path. 
You can find this at _JAVA_LOCATION_\jdk-19\bin.

To launch FlatLand 1.0 alpha run 'FlatLand1.0a.exe'.
Please reference the user guide for more information.

This software is in the very early stages of development, you will likely encounter issues with the system.
If you have the time please submit any issues encountered somewhere in the survey.
